# Atividade — Paginação Simples DAW3

## Requisitos

- Node.js 18+
- npm

## Como rodar

```bash
# 1. Instalar dependências
npm install

# 2. Rodar em desenvolvimento
npm run dev
```

Acesse: [http://localhost:3000](http://localhost:3000)

## Estrutura de páginas

| Rota | Descrição |
|------|-----------|
| `/` | Página inicial com lista de professores |
| `/professor/1` | Página do Prof. Carlos Mendonça |
| `/professor/2` | Página da Profa. Ana Paula Lima |
| `/professor/3` | Página do Prof. Ricardo Souza |
